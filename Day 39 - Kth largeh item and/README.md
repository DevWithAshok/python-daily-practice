# Day 39 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Kth Largest Number
- Accepts a list of numbers as input.
- Finds the Kth largest element in the list.
- Prints the required largest number based on the given position.

### 2. Extract Digits
- Accepts a string as input.
- Extracts all numeric digits present in the string.
- Calculates and prints:
  - Sum of all extracted digits
  - Smallest digit
  - Largest digit

## Concepts Practiced
