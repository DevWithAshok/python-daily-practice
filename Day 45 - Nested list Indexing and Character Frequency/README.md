# Day 45 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Nested List Indexing
- Uses a nested tuple structure containing multiple groups of numbers.
- Accepts a number as input.
- Searches for the number within the nested tuples.
- Prints the row index and column index where the number is found.

### 2. Character Frequency
- Accepts a string as input.
- Converts all characters to lowercase.
- Ignores spaces while processing.
- Counts the frequency of each character.
- Prints each character along with its occurrence count in sorted order.

## Concepts Practiced
- Nested Data Structures
- Tuple Indexing
- Loops
- String Manipulation
- Sets
- Character Frequency Analysis
