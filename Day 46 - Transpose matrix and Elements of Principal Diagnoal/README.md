# Day 46 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Transpose Matrix
- Accepts the dimensions and elements of a matrix as input.
- Converts rows into columns and columns into rows.
- Prints the transpose of the given matrix.

### 2. Elements of Principal Diagonal
- Accepts the dimensions and elements of a square matrix as input.
- Extracts the elements present on the principal diagonal.
- Prints the diagonal elements as a list.
