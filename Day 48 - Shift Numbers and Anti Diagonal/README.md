# Day 48 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Shift Numbers
- Accepts a sequence of numbers as input.
- Shifts the elements according to the given logic.
- Displays the modified sequence after shifting.

### 2. Anti Diagonal of a Matrix
- Accepts the dimensions and elements of a matrix.
- Extracts the elements present on the anti-diagonal.
- Prints the anti-diagonal elements of the matrix.
