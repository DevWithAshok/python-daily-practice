# Day 60 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Rotate D Times
- Accepts a list of elements and a rotation count (D) as input.
- Rotates the elements of the list by D positions.
- Displays the updated list after performing the rotation.
- Demonstrates list manipulation and indexing techniques.

### 2. N Greatest Numbers
- Accepts a list of numbers and a value N as input.
- Identifies the N largest numbers in the list.
- Displays the selected numbers in the required format.
- Demonstrates sorting and list processing techniques.

## Concepts Practiced
- Lists
- List Rotation
- Sorting
- Indexing
- Input Processing
