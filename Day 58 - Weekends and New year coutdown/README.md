# Day 58 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Weekends
- Accepts a range of dates or a specific period as input.
- Identifies all weekend days within the given range.
- Displays the weekends found in the specified duration.

### 2. New Year Countdown
- Accepts the current date and time.
- Calculates the remaining time until New Year's Day.
- Displays the countdown in a readable format.
