# Day 36 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Sum of Cubes from M to N
- Accepts two integers, M and N, as input.
- Calculates the cube of each number in the range from M to N (inclusive).
- Prints the sum of all cubes in the given range.

### 2. Number of Cars Needed
- Accepts the number of people as input.
- Calculates the minimum number of cars required.
- Assumes each car can accommodate up to 5 people.
- Prints the total number of cars needed.
