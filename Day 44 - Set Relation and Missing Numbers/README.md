# Day 44 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Set Relation
- Accepts a set of numbers as input.
- Compares the input set with a predefined set.
- Determines whether the input set is:
  - A Subset
  - A Superset
  - A Disjoint Set
- Prints the corresponding relationship.

### 2. Missing Numbers
- Accepts a sequence of numbers as input.
- Identifies the numbers missing within the range from 1 to the largest number in the sequence.
- Stores and prints all missing numbers in a list.
