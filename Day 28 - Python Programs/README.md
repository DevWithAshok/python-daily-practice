## Program 1 - Temperature Conversion

### Description

This program converts temperature values between:

* Celsius
* Fahrenheit
* Kelvin

based on the input unit.

### Concepts Used

* Conditional statements
* Arithmetic operations
* String handling
* Type conversion

---

## Program 2 - Sum of N Terms in Harmonic Series

### Description

This program calculates the sum of harmonic series up to N terms.

Formula:

1 + 1/2 + 1/3 + ... + 1/N

### Concepts Used

* Loops
* Floating point arithmetic
* Mathematical series
* Accumulator pattern
