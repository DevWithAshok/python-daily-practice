# Day 40 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Sum of N Harmonic Series
- Accepts a positive integer N as input.
- Uses recursion to calculate the sum of the harmonic series:
  - 1 + 1/2 + 1/3 + ... + 1/N
- Prints the result rounded to two decimal places.

### 2. Palindrome Checker
- Accepts a string as input.
- Checks whether the string reads the same forwards and backwards.
- Returns and prints `True` if it is a palindrome, otherwise `False`.

## Concepts Practiced
