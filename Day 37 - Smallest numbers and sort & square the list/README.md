# Day 37 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Sort and Square the List
- Accepts a list of numbers separated by commas.
- Calculates the square of each number.
- Sorts the squared values in ascending order.
- Prints the resulting list.

### 2. Smallest Number in a List
- Accepts a list of numbers separated by commas.
- Finds the smallest number in the list.
- Prints the smallest value.
