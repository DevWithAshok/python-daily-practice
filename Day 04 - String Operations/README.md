# Day 04 - String Operations in Python

## Topic

String slicing, repetition, and length

## Code

```python id="h7t3kp"
word = input()
print(word[::-1])

word = "*"
print(word * 2)
print(word * 2)

word = input()
print(len(word))
```

## Explanation

This program demonstrates:

* Reversing a string
* Repeating characters
* Finding string length

## Key Concepts

### Slicing

`[::-1]` is used to reverse a string.

### String Multiplication

`*` is used to repeat a string multiple times.

### len()

Returns the number of characters in a string.

## Example

Input:
Hello

Output:
olleH
