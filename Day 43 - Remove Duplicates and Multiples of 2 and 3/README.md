# Day 43 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Remove Duplicates
- Accepts a list of numbers as input.
- Removes duplicate elements from the list.
- Sorts the unique elements in ascending order.
- Prints the resulting list.

### 2. Multiples of 2 and 3
- Accepts an integer N as input.
- Generates multiples of 2 and multiples of 3 up to N terms.
- Finds:
  - Numbers that are multiples of 2 but not multiples of 3.
  - Numbers that belong to either set but not both (symmetric difference).
- Prints the resulting sorted lists.
