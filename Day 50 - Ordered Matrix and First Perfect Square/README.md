# Day 50 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Ordered Matrix
- Accepts matrix elements as input.
- Processes and arranges the matrix according to the required order.
- Displays the resulting ordered matrix.
- Demonstrates matrix traversal and manipulation techniques.

### 2. First Perfect Square
- Accepts a sequence of numbers as input.
- Identifies the first perfect square number in the sequence.
- Prints the first perfect square if present.
- Demonstrates number property checking and searching.

## Concepts Practiced
- Matrices
- Nested Lists
- Loops
- Conditional Statements
