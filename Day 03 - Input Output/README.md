# Day 03 - Input and Output in Python

## Topic

Taking user input and displaying output

## Code


```python id="y7t2kp"
word1 = input()
word2 = input()

print(word1)
print(word2)

word = input()
print("Given input: " + word)
```

## Explanation

This program demonstrates:

* Taking input from the user using input()
* Printing multiple inputs
* Concatenating strings

## Key Concepts

### input()

Used to take input from the user.

### String Concatenation

Using + to combine strings.

## Example

Input:
Hello
World

Output:
Hello
World

Given input: Hello

* Practiced string output
