# Day 38 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Mean of a List
- Accepts a list of numbers separated by commas.
- Calculates the arithmetic mean (average) of the numbers.
- Prints the mean value.

### 2. Median of a List
- Accepts a list of numbers separated by commas.
- Sorts the numbers in ascending order.
- Calculates the median:
  - For an odd number of elements, the middle value is returned.
  - For an even number of elements, the average of the two middle values is returned.
- Prints the median value.
