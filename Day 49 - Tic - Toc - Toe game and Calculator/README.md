# Day 49 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Calculator
- Accepts two numbers and an arithmetic operator as input.
- Performs the specified operation:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Modulus (%)
- Prints the calculated result.

### 2. Tic Tac Toe Game
- Accepts a 3×3 Tic Tac Toe board as input.
- Checks rows, columns, and diagonals for a winning combination.
- Determines whether:
  - Abhinav Wins (O)
  - Anjali Wins (X)
  - Tie
- Prints the game result.
