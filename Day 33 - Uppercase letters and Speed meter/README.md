# Day 33 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Speed Meter
- Accepts the speed of a vehicle as input.
- Displays:
  - Normal for speeds below 60.
  - Warning for speeds between 60 and 79.
  - Over Speed for speeds 80 and above.

### 2. Count Uppercase Letters
- Accepts a word or string as input.
- Counts and prints the number of uppercase letters present.

## Concepts Practiced
- Functions
- Conditional Statements
- String Operations
- Character Case Checking
- Loops
