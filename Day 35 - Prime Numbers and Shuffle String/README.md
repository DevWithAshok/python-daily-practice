# Day 35 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Prime Numbers from 1 to N
- Accepts a number N as input.
- Checks each number from 1 to N for primality.
- Prints all prime numbers within the given range.

### 2. Shuffle String
- Accepts a string and a list of indices as input.
- Rearranges the characters of the string according to the given indices.
- Prints the shuffled string.
