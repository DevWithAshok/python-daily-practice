# Day 34 - Python Practice 

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. ATM PIN Verification
- Accepts a PIN code as input.
- Checks whether the PIN contains only digits.
- Validates if the PIN length is either 4 or 6 digits.
- Displays whether the PIN code is valid or invalid.

### 2. Calculate Bill
- Accepts the bill amount as input.
- Applies discounts based on the amount:
  - 5% discount for amounts below 500.
  - 10% discount for amounts from 500 to 2499.
  - 20% discount for amounts 2500 and above.
- Prints the final payable amount after discount.
