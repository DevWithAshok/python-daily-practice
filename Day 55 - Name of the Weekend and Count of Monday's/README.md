# Day 55 - Python Practice

This repository contains two Python practice programs completed as part of my daily Python learning journey.

## Programs

### 1. Name of the Weekend
- Accepts a date as input.
- Determines whether the date falls on a weekend.
- Displays the name of the weekend day (Saturday or Sunday) when applicable.

### 2. Count of Mondays
- Accepts a month and year as input.
- Calculates the total number of Mondays in the specified month.
- Prints the count of Mondays.
